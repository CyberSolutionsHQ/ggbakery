# Golden Gulf Bakery Static Site

GitHub Pages-ready static website for Golden Gulf Bakery.

## Tech

- HTML5
- CSS3
- Vanilla JavaScript
- No build step

## Brand Direction

Palette is defined in `css/styles.css`:

- Deep Gulf Blue: `#0D3B66`
- Ocean Blue: `#1F5F99`
- Golden Crust: `#E3A92B`
- Warm Bread Tan: `#F2D6A2`
- Cream Flour White: `#FBF8F2`

Typography uses Google Fonts:

- Headings: Lora (serif)
- Body: Source Sans 3 (sans-serif)

## Deploy on GitHub Pages

1. Push to your GitHub repository.
2. In repository Settings > Pages, set source to deploy from branch.
3. Keep the file/folder structure as-is so relative links continue to work in project-site subpaths.
4. Update `sitemap.xml` and `robots.txt` with your real GitHub Pages URL.

## Notes

- Special-order form generates a `mailto:` draft to `goldengulfbakery@gmail.com`.
- Site includes Mississippi cottage food notice and availability messaging.
- No analytics scripts are enabled by default.
